# TDS
C++, sfml

Inputs

        [W] [A] [S] [D] to Move
        [Left Mouse Button] to Shoot

        Scroll Mouse Wheel to Zoom in and out
        Move Mouse to Look Around

        Press numbers from [1] to [4] to change the weapon

        [Esc] to Close Window

        // Debug
        [F1] to Set FPS Limit to 10 FPS
        [F2] to Set FPS Limit to 300 FPS
